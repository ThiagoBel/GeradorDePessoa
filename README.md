# Gerador De Pessoa.

## Como funciona?
- Ele cria uma pessoa com QI aleatorios e linguagens de programações que a pessoa sabe.

## Características
- Nome & Sobrenome
- Idade
- Nacionalidade
- Linguagem de programação preferida

## Como executar?
- Tenha Lua instalado
- Use o comando: ```Lua pessoaGerador.lua``` no terminal para executar.
